## **git remote add**

**git remote add *[Создание-записи-о-подключении-к-репозиторию]*** - Данная команда, используется для создания записи о новом подключении к удалённому репозиторию.

Чтобы выполнить данную команду, требуется ввести:

```bash-
git remote add
```
