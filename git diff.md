## ***git diff***

**git diff *[Вычисление_разницы]*** - используется для вычисления разницы между любыми двумя Git источниками данных **(По типу, индексов и коммитов)**.

Чтобы найти разницу в Git, используется команда:

```bash-
git diff
```
